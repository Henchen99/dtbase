Environment

Continuous Improvement:

  • Ongoing process of reducing environmental impact through design iterations
  • Example: LED lightbulbs using 75% less energy than incandescent versions
  • Example: Smart thermostats learning user patterns to optimize heating
  • Lifecycle analysis used to identify improvement areas (raw materials → disposal)

Efficient Working:

  • Minimising waste in production processes (energy, materials, time)
  • Example: Toyota's "just-in-time" manufacturing reducing warehouse needs
  • Example: 3D printing producing less material waste than subtractive methods
  • Lean manufacturing principles eliminating unnecessary steps

Pollution:

  • Direct and indirect contamination from production/use/disposal
  • Example: Microplastics from synthetic textiles entering waterways
  • Example: E-waste containing heavy metals leaching into soil
  • Air pollution from manufacturing (CO₂, VOCs, particulate matter)

Global Warming:

  • Greenhouse gas emissions throughout product lifecycles
  • Example: Fast fashion contributing 10% of global carbon emissions
  • Example: Data centers for digital products using massive electricity
  • Transportation impacts (sea freight vs air freight emissions)

Positive Case Study: Interface carpet tiles using recycled fishing nets (Net-Works program)

Negative Case Study: Planned obsolescence in electronics creating waste mountains
